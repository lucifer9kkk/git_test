##### **Command Line Code -**



|Code|To open VS Code|
|-|-|
|Code \[file Name] |To Open a VS code file|

|cd \[Tab] then type folder name and hit \[Tab]|For open any folder|
|-|-|
|cd and hit enter|For Going back to Home|
|cd \~/\[Folder Name]|For Moving for Diff Folder|
|cd \[Folder name or TAB]|For going into folder's folder|
|cd ..|For going into previous folder|

|**mkdir \[Folder Name]**|**To make a new folder**|
|-|-|
|rmdir \[Folder Name]|To remove a folder|
|touch \[File Name]|To Make a new file|
|rm \[File Name]|To remove any file|

|**mv \[File Name] \[folder Name/]**|**To move a file in different folder**|
|-|-|
|mv \[old file name] \[New file Name]|To rename a folder or file use|
